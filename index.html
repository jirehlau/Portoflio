GuyBean

GuyBean is a restaurant booking platform offering off peak hour dine ins on selected food items. I first planned by drawing out an ERD diagram to understand what type of relationships I would need for my back-end and database. Then prototyped using Figma, implemented the backend functions using model, views, controllers, and eventually integrated some of the front-end work with EJS and CSS.
